Customizing

This shows you how tou can customize fossajs, without edditing fossa.js.

          
            var fossa = new fossa({
              title: 'custom',
              struc: 'custom',
              custEl: '<p>',
              cClass: 'customEl',
              divText: 'This is a custom element'
            });
          
        
          
            <head>
              <title>custom</title>
            </head>
            <body>
              <p class="customEl">This is a custom element</p>
            <body>

This mainly allows you to add your own custom elements using these properties: struc, custEl, CClass, and divText. Using the same value as in struc. You need to have 'custom' in the struc property to tell the javascript that it is a custom structure.
