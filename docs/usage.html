Usage

If you looked at the example in the getting started section, there wasn't much to it. Here I explain what was going on, If you want to understand the actual javascript, go here and read the comments.

Basic Properties

Fossa is object oriented, so that means that we are going to be using and object constructor.

        
          var fossa = new fossa({
            //The properties go here.
          });

Now, here are all of the properties in one constructor.

          
            var fossa = new fossa({
              title: 'list',
              struc: 'custom',
              divText: 'A little list of properties'
              divBackground: '#f5f5f5'
              headerText: 'A list',
              optForHeader: 'y',
              pad: '0',
              mar: '0',
              custEl: '<ul>',
              cClass: 'list'
            });

These are all of the main options for now.

Explanation

I will give examples to explain what is going on. Checkout the html box to see what happens

            
              var fossa = new fossa({
                title: 'Demo',
                struc: 'strict',
                divText: '<p>This is a cool cat demo</p>'
              });
          
        
          
            <head>
              <title>Demo<title>
            </head>
            <body>
              <div id="div1">
                <div id="div2">
                  <p>
                    This is a cool cat demo
                  </p>
                <div>
              <div>
              <script src="https://unpkg.com/fossajs"><script>
            </body>

This is mainly what your HTML should look like when you inspect element. The title property edits the title tag, the struc property edits the way that the divs are placed. For struc, there are 4 options, 'strict', 'loose', 'sagging', and 'custom'. They are basically self explanitory. Then, the text property, puts what ever you type in between the divs.
