Usage

If you looked at the example in the getting started section, there wasn't much to it. Here I explain what was going on, If you want to understand the actual javascript, go here and read the comments.

Basic Properties

Fossa is object oriented, so that means that we are going to be using and object constructor.

        
          var fossa = new fossa({
            //The properties go here.
          });

Now, here are all of the properties in one constructor.

          
            var fossa = new fossa({
              title: 'list',
              struc: 'custom',
              divText: 'A little list of properties'
              divBackground: '#f5f5f5'
              headerText: 'A list',
              optForHeader: 'y',
              pad: '0',
              mar: '0',
              custEl: '<ul>',
              cClass: 'list',
            });
