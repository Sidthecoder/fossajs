Getting Started

After you have installed it you want to create a new object constructor. We will label the var name "app" and put the constructor inside of it.

Your full code should look like this

        
          var fossajs = require('fossajs');
          
          var app = new fossa({
            //The code goes here
          });

This will give you the base for creating the page structure

Example

This will show how the basic build will look, this is explained more in the next section.

        
          var fossajs = require('fossajs');
          
          var app = new fossa({
            title: 'Example',
            struc: 'strict',
            divText: 'Cats are cool',
            divBackground: '#f5f5f5'
          });

These are not all of the object items that can be use, they are just some of them.
