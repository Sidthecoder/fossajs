Installation

It is quite easy to install Fossa. The easiest way is to use UNPKG.

        
          <script src="https://unpkg.com/fossajs"></script>

Or you can use npm, if that is what you prefer.

        
          $ npm install --save-dev fossajs

Then you have to require it using a variable

es6:

        
          const fossajs = require('fossajs');

es5:

        
          var fossajs = require('fossajs');

This is all you need to get started, speaking of getting started, go on over to the getting started section!
